Moment 1 - Laborationsuppgift

I den här uppgiften ska nodeJs användas för att automatisera och underlätta arbetsprocessen när man utvecklar en webbplats tillsammans med ett kommandobaserat verktyg. Utvecklingsmiljön automatiseras baserat på Git och NodeJs, publiceringen kan då ocskå göras automatiskt vid uppdatering av Git-repot. Genom det kommandobaserade verktygen kan man sedan automatisera transpileringen och minifieringen av CSS, HTML och JavaScript samt automatiskt optimera grafikfiler. Resultatet av uppgiften kan sedan användas som en broilerplate inför framtida projekt med webbutveckling.
